What is the absolute neutrophil count calculator?
The absolute neutrophil count calculator is a tool that calculates the total count of neutrophils in the blood.
How to calculate absolute neutrophil count?
To calculate absolute neutrophil count you should use the following formula:
absolute neutrophil count = white blood cells * (neutrophil granulocytes + neutrophilic band forms) / 100
